# mhw1
Primo progetto corso WEB PROGRAMMING

## Nome:
Dino's Cinema web.

## Descrizione: 
Il progetto verte sullo sviluppo di un sito web di una catena di cinema: "Dino’s Cinema’". Il sito contiene diversi collegamenti ad altre pagine che conterranno le informazioni sui dati che sono stati organizzati e sono manipolabili tramite il database realizzato al primo semestre (collegamento alle informazioni sulle sale, sedi, film e quindi i relativi generi, staff..). L’idea è quella di mostrare una serie di dati pubblici e varie informazioni ai visitatori della pagina, e mantenere dei dati ‘’privati’’ per il personale. Tramite il sito è possibile infatti effettuare un ‘’Log in’’, il personale autorizzato effettuando l’accesso avrà a disposizione gli ulteriori dati del database, non visibili ai clienti/visitatori.

## Stato del progetto:
Dopo una settimaa di duro lavoro e studio del materialle didattico messo a disposizione, la consultazione di numerosi blog online, sono riuscito a realizzare il layout complessivo del sito. Il sito e' in fase di svilupppo, i collegamenti <a> non contengono alcun riferimento, le immagini sono linkate.. 

## Utilizzo:
Il sito non è ancora online, ma e' possibile vederne una anteprima grazie ai file allegati.
